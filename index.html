important note on windows defender

because Apex Autoclicker is a brand-new, independently developed application, it does not have a multi-thousand-dollar microsoft publisher certificate. therefore, windows will likely flag the installer.

furthermore, to allow you to start and stop the clicker while you are inside a video game, Apex uses global keyboard hooks to listen for your hotkey in the background. windows defender and smartscreen often flag this behavior as suspicious.

this is a false positive. Apex is 100% safe, and nobody but NoKDR has the source code.
